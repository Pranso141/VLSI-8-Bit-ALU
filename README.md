# VLSI-8-Bit-ALU
Making a 4 bit ALU to practice Verilog Concepts

I aim for the ALU to be able to execute the following tasks:
#Arithmetic Tasks
1. 8 Bit Carry Look Ahead Addition
2. 8 Bit Carry Look Ahead Addition with carry
3. 8 Bit Subtraction
4. 8 Bit Subtraction with Borrow
5. 2's Complement
6. Increment
7. Decrement
#Bitwise Operations
1. AND
2. OR
3. XOR
#Bit Shift Operations
1. Aritmethic Shifitng
2. Logical Shifting
3. Rotate
4. Rotate through Carry
